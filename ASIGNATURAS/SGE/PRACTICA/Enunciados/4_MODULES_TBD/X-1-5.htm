Step 1: Setup Development Environment

- Odoo modules are stored in a custom directory

- Ensure you have a custom directory to store your module

mkdir -p /opt/odoo/custom_addons

cd /opt/odoo/custom_addons

- Ensure that this directory is accessible by the Odoo user

sudo chown -R odoo:odoo /opt/odoo/custom_addons

- Update the Odoo configuration file (/etc/odoo/odoo.conf) to include this path

sudo nano /etc/odoo/odoo.conf

- Locate the addons_path line and modify it

addons_path = /opt/odoo/custom_addons,/usr/lib/python3/dist-packages/odoo/addons

- Save and exit

- Restart Odoo

sudo systemctl restart odoo

Step 2: Create the Module Skeleton

- Navigate to the custom modules directory

cd /opt/odoo/custom_addons

- Use the Odoo scaffold command to generate a basic module structure

/odoo/odoo-bin scaffold my_product_module /opt/odoo/custom_addons

- Navigate into the module

cd my_product_module

Step 3: Define the Module Manifest

- Edit the __manifest__.py file

nano __manifest__.py

- Modify it as follows

{

'name': 'My Product Module',

'version': '1.0',

'summary': 'Manage product entities with CRUD operations',

'author': 'Your Name',

'category': 'Inventory',

'license': 'LGPL-3',

'depends': ['base'],

'data':

[

'security/ir.model.access.csv',

'views/my_product_entity_view.xml',

],

'installable': True,

'application': True,

}

Step 4: Define the Data Model

- Edit the models directory and create a file my_product_entity.py

mkdir models

nano models/my_product_entity.py

- Add the following content

from odoo import models, fields

 

class MyProductEntity(models.Model):

_name = 'my_product_entity'

_description = 'My Product Entity'

 

code = fields.Char(string='Code', required=True)

name = fields.Char(string='Name', required=True)

price = fields.Float(string='Price', required=True)

Step 5: Define Security Rules

- Create the security directory and define access rights

mkdir security

nano security/ir.model.access.csv

- Add the following

id,name,model_id:id,group_id:id,perm_read,perm_write,perm_create,perm_unlink access_my_product_entity,my_product_entity,model_my_product_entity,,1,1,1,1
