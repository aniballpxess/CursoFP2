Step 6: Define Views

Create the views directory and define a form, tree, and menu for the model:

mkdir views

nano views/my_product_entity_view.xml

Add:

<odoo>

    <record id="view_my_product_entity_tree" model="ir.ui.view">

        <field name="name">my.product.entity.tree</field>

        <field name="model">my_product_entity</field>

        <field name="arch" type="xml">

            <tree>

                <field name="code"/>

                <field name="name"/>

                <field name="price"/>

            </tree>

        </field>

    </record>

 

    <record id="view_my_product_entity_form" model="ir.ui.view">

        <field name="name">my.product.entity.form</field>

        <field name="model">my_product_entity</field>

        <field name="arch" type="xml">

            <form>

                <sheet>

                    <group>

                        <field name="code"/>

                        <field name="name"/>

                        <field name="price"/>

                    </group>

                </sheet>

            </form>

        </field>

    </record>

 

    <menuitem id="my_product_menu" name="My Product Module" sequence="1"/>

 

    <menuitem id="my_product_entity_menu" name="Products"

              parent="my_product_menu" action="action_my_product_entity"/>

 

    <record id="action_my_product_entity" model="ir.actions.act_window">

        <field name="name">My Product Entity</field>

        <field name="res_model">my_product_entity</field>

        <field name="view_mode">tree,form</field>

    </record>

</odoo>

Step 7: Initialize the Module

Edit __init__.py:

nano __init__.py

Modify it to include:

from . import models

Then, edit models/__init__.py:

nano models/__init__.py

Add:

from . import my_product_entity

Step 8: Install the Module in Odoo

Restart the Odoo server:

sudo systemctl restart odoo

Enable developer mode by adding ?debug=1 at the end of the Odoo URL:

http://localhost:8069/web?debug=1

  • Go to Apps.
  • Click Update Apps List.
  • Search for My Product Module.
  • Click Install.

Step 9: Test CRUD Operations

  • Navigate to My Product Module > Products.
  • Try creating, editing, and deleting a product entity.
